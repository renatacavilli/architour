# Architour Frontend

Aquest repositori conté el frontend del projecte **Architour**, una pàgina web organitzada en carpetes per idioma, rutes, viatges, imatges i fitxers JavaScript.

**Architour** és una pàgina web dedicada a descobrir arquitectura, urbanisme i espais emblemàtics a través de rutes guiades i contingut visual interactiu.

El projecte presenta diferents itineraris per barris i ciutats, amb una navegació intuïtiva, contingut multimèdia i suport multilingüe.


## Estructura del projecte

- `ca/` — versió en català
- `cast/` — versió en castellà
- `en/` — versió en anglès
- `imatges/` — recursos d'imatge
- `java/` — fitxers JavaScript
- `index.html` — pàgina principal

## Funcionalitats

- Navegació per idiomes
- Menú responsive
- Galeries i contingut per diferents rutes
- Pàgines específiques per barris i ciutats
